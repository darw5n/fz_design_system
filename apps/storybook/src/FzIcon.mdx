# FzIcon

Design system icon component built on Font Awesome. Renders a single icon with configurable size and variant. To apply design system text color, use the <code>v-color</code> directive on the component (see Coloring with v-color below).

## Features

- Font Awesome icon set (Pro, via icon kits)
- Configurable size (xs, sm, md, lg, xl, 2xl) and variant (fas, far, fal, etc.)
- Color via <code>v-color</code> directive (requires @fiscozen/style and <code>setupFzStyle</code>)
- Spin animation support

## Installation

```bash
npm install @fiscozen/icons
```

To use <code>v-color</code> for icon color, also install and register <code>@fiscozen/style</code> (see Typography documentation).

## Basic Usage

```vue
<script setup>
import { FzIcon } from '@fiscozen/icons'
</script>

<template>
  <FzIcon name="check" />
  <FzIcon name="bell" size="lg" />
  <FzIcon name="check" v-color:blue />
  <FzIcon name="check" v-color:blue="500" />
</template>
```

## Coloring with v-color

Apply design system text color by using the <code>v-color</code> directive on <code>FzIcon</code>. The directive runs on the component root (<code>&lt;p role="presentation"&gt;</code>), and the icon inherits the color via <code>currentColor</code>.

**Syntax:** <code>v-color:colorName</code> (default weight) or <code>v-color:colorName="weight"</code> (e.g. <code>v-color:blue="500"</code>, <code>v-color:semantic-error="200"</code>).

For the full list of supported colors (base, semantic, core) and default weights, see [Typography â€“ v-color](../?path=/docs/foundations-typography--docs#v-color).

**Example:**

```vue
<template>
  <FzIcon name="check" v-color:blue />
  <FzIcon name="check" v-color:blue="500" />
  <FzIcon name="alert-circle" v-color:semantic-error />
  <FzIcon name="info" v-color:semantic-info="300" />
</template>
```

## Accessibility

FzIcon renders a decorative icon by default (no aria-label). The root <code>&lt;p role="presentation"&gt;</code> ensures the wrapper is not announced as a paragraph. When the icon conveys meaning, wrap it in a button or link and provide an accessible label on that control (e.g. <code>FzIconButton</code> with <code>ariaLabel</code>).
