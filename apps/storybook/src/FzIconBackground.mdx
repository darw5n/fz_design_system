# FzIconBackground

Wrapper around FzIcon that adds a background to the icon container. Use it when you need an icon with a visible background (e.g. status indicators, badges, or highlighted icons). It accepts the same props as FzIcon.

## Features

- Same API as FzIcon (name, size, variant, spin) plus <code>backgroundColor</code>
- Configurable background on the icon container (rounded, padded)
- Color via <code>v-color</code> directive (requires @fiscozen/style and <code>setupFzStyle</code>)
- Font Awesome icon set (Pro, via icon kits)

## Installation

```bash
npm install @fiscozen/icons
```

To use <code>v-color</code> for icon color, also install and register <code>@fiscozen/style</code> (see Typography documentation).

## Basic Usage

```vue
<script setup>
import { FzIconBackground } from '@fiscozen/icons'
</script>

<template>
  <FzIconBackground name="check" />
  <FzIconBackground name="bell" size="lg" />
  <FzIconBackground name="check" v-color:blue="500" />
</template>
```

## Coloring with v-color

You can apply the design system text color to <code>FzIconBackground</code> with the <code>v-color</code> directive, same as <code>FzIcon</code>. The directive targets the inner icon root, and the icon inherits the color via <code>currentColor</code>.

**Syntax:** <code>v-color:colorName</code> or <code>v-color:colorName="weight"</code> (e.g. <code>v-color:blue="500"</code>). For supported colors and weights, see [Typography â€“ v-color](../?path=/docs/foundations-typography--docs#v-color).

```vue
<template>
  <FzIconBackground name="check" v-color:blue />
  <FzIconBackground name="alert-circle" v-color:semantic-error="200" />
</template>
```

## Examples

### Sizes

```vue
<template>
  <div class="flex items-center gap-4">
    <FzIconBackground name="bell" size="xs" />
    <FzIconBackground name="bell" size="sm" />
    <FzIconBackground name="bell" size="md" />
    <FzIconBackground name="bell" size="lg" />
  </div>
</template>
```

### Custom background

```vue
<template>
  <FzIconBackground name="check" backgroundColor="core-white" />
  <FzIconBackground name="bell" backgroundColor="grey-100" />
  <FzIconBackground name="alert-circle" backgroundColor="blue-500" />
</template>
```

## Accessibility

Same as FzIcon: the icon is decorative by default. When it conveys meaning, use it inside a button or link with an accessible label.

## Related Components

- **FzIcon**: Base icon component without background; use when you only need the icon.
- **FzIconButton**: Icon-only button; use when the icon is actionable.
